XMeme

Source Code

XMEME is a meme sharing platform where users can see and search memes based on captions or owner names. Registered users can post/share memes. Only admins have the authority to delete/edit memes and even block/deactivate users.

QEats

Source Code

QEats is a food ordering app where users can view the list of restaurants within 3km - 5km radius. Restaurants can be searched based on restaurant names, attributes or the food items they serve. Items can be added/removed to/from the cart and then a user can place order(s).

QMoney

Source Code

QMoney is a portfolio analyzer that uses third party APIs like Tiingo and AlphaVantage to get the list of stocks and calculate the annualized returns using the open/close price of the stocks.

Cell Counter

Source Code

A MATALAB application that uses digital image processing techniques, capable of counting the Red Blood Cells (RBCs), White Blood Cells (WBCs) and Platelets respectively from a colored image of a peripheral blood. smear/ film (PBF).
